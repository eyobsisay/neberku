How It Works
  1. Token Validation: The system checks JWT token structure and expiration
  2. API Interceptor: All API requests are intercepted for auth errors
  3. Error Detection: Detects 401, 403, and specific JWT error messages
  4. Automatic Redirect: Clears tokens and redirects to login page
  5. User Notification: Shows friendly error message before redirect
